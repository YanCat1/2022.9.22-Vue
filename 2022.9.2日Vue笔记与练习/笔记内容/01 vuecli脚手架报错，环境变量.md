我们在安装vue cli之后，会显示不是内部或者外部的指令

需要进行环境变量的添加

cmd 中输入 **npm config list ** 找到 prefix 属性 根据地址找到

prefix = "C:\\Users\\Administrator\\AppData\\Roaming\\npm"



右键 此电脑 => 点击属性 => 高级系统设置 => 环境变量 

点击 系统变量中的 path => 编辑 => 新建 

将路径地址复制确定即可
